Island Layout — Where does everything go?

How the canvas world is arranged, and how the React UI panels sit with it. Bottle path shown in dashed amber.

A — Horizontal Beach Path

Beach runs the full bottom edge. Villagers spaced left→right: Town Hall → Forest → Library → Coffee Shop → Mailbox. Bottle travels horizontally. Workflow panel floats top-right.

Natural reading direction matches task flow. Most like classic AC top-down beach.

B — Canvas Left, Panel Right

Island canvas takes ~70% of window width. Workflow panel is a persistent right sidebar. Task input sits below the sidebar. Clean separation: world on left, status on right.

Like having a game window and a HUD panel. Very readable, never overlaps content.

C — Full Canvas + Floating Panels

Canvas is full bleed — fills the entire window. React UI panels float over the island as glass cards: task input top-left, workflow status top-right. Island world always visible.

Most immersive. The island is the app. Panels feel like AC thought bubbles.
